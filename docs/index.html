Mondlandung

Ziel des Projekts

Ziel dieses Projekts ist es zu simulieren, wie ein Raketenstart von der Erde den Mond erreichen und auf diesem weiche Landen kann. Diese Simulation beinhalten unter anderem die Gravitation aller Objekte, eine Massenabnahme der Rakete und Beschleunigungsvektoren in 3-D für die Objekte.

Um eine Rakete abschiessen zu können sind verschiedene Startparameter wie

  • Raketentreibstoff
  • Abschusszeitpunkt
  • Abschussrichtung

notwenig.

Dem User soll es überlassen werden, ob er diese Parameter selber definiert oder diese Berechnen lassen möchte. Wird eine Berechnung angestossen wird ein lineares Gleichungssystem im Hintergrund gelöst. Die Lösung dazu wird grafisch festgehalten.

Änderung

Nach dem Meeting vom 27. April mit Herr Heuberger war klar, dass wir unser Ziel der weichen Landung anpassen müssen. Eine weiche Landung hätte eine stabile Umlaufbahn um den Mond erfordert und dann müsste sich ein Landemodul lösen, welche mithilfe von Bremskraftriebwerken laden würde. Dies erachtet Herr Heuberger als zu aufwendig für den Zeitrahmen dieses Projekts und es wurde beschlossen, dass "nur" eine stabile Umlaufbahn erreicht werden muss. Um das lineare Gleichungssystem zu lösen wurde ein randomisierter Ansatz gewählt. Der Algorithmus wählt die Startparameter nun zufällig und bewertet dann die Qualität der Lösung. Dies führt nun dazu dass das Gleichungssystem z.B. 1000 mal durchlaufen wird. Um zu zeigen, dass der Algorithmus im Hintergrund arbeitet wurde beschlossen, dass z.B. jeder hundertste Versuche grafisch dargestellt wird.

Position x(t)

Geschwindigkeit v(t)

Rakete

Position xErde = [40; 40; 40]

Startposition xRakete0 = [45; 40; 40]

Startvektor uRakete (z.B. [0; 1; 0])

Startzeitpunkt t0 = 0

Masse Erde me = 600

Masse Brennstoff mb = 10

Masse Rakete mr = 10 + mb = 20

Beschleunigung bRakete = [0; 2; 0]

xRakete(0) = xRakete0;

xRakete(t) = xRakete0 + uRakete * vRakete(t) * t

vRakete(t) = aRakete(t) * t

aRakete(t) = bRakete - G * (mr*me / (xRakete(t-1) - xErde))

vRakete(t) = bRakete - G * (mr*me / (xRakete(t-1) - xErde)) * t

Mond

Startposition xMond0 = [20; 40; 40]

Masse Mond mm = 200

aMond0 = [0; 10; 0]

xMond(t) = xMond0 + [0; 1; 0] * (kMond) * t

vMond(t) = aMond(t) * t

aMond(0) = aMond0

aMond(t) = aMond(t-1) - G * (mm*me / (xMond(t-1) - xErde))
