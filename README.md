# Mondlandung

## Installation

* `git clone git://github.com/fabian/Mondlandung.git`
* `open test.html`

## Testing

* Get the latest Node installer from [nodejs.org](http://nodejs.org/). You need Node 0.6.3 or newer.
* `npm install -g buster`
* `buster server`
* `open http://localhost:1111`
* `buster test`
